### Actions

Get user handler

```json
{ "action": "getUserHandler", "body": "{ \"userName\": \"userName1\" }" }
```

Create user handler

```json
{
  "action": "createUserHandler",
  "body": "{ \"userName\": \"userName1\" }"
}
```

Increment user handler

```json
{
  "action": "increaseCounterHandler",
  "id": "e4986bf9-40d5-4d6b-ae1b-986d32350bae",
  "incrementValue": 20
}
```
