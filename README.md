### Actions

Get user handler

```json
{ "action": "getUserHandler", "id": "a6672561-e491-4a76-948a-fa7c7d405c0a" }
```

Create user handler

```json
{
  "action": "createUserHandler",
  "body": "{\"name\": \"user1\"}"
}
```

Increment user handler

```json
{
  "action": "increaseCounterHandler",
  "id": "a6672561-e491-4a76-948a-fa7c7d405c0a",
  "incrementValue": 20
}
```
